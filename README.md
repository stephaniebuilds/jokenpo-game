# 🎮 Jokenpô Modern JS - Interactive Game

A stylish and functional Rock Paper Scissors game featuring a modern "Soft UI" aesthetic. This project focuses on DOM manipulation, CSS effects (glow and glassmorphism), and game logic to provide a clean and engaging user experience.

## ✨ Features

* **Modern UI Design:** Clean interface using a soft color palette, subtle shadows, and a sophisticated glow effect on the title.
* **Real-Time Scoring:** Dynamic score tracking for both the player and the machine, updated instantly after each round.
* **Visual Feedback:** Clear status messages (e.g., "Rock breaks Scissors") in a refined gray tone to contrast with the vibrant primary UI elements.
* **Interactive Controls:** Large, intuitive emoji buttons with hover animations and lift effects for a tactile feel.
* **Fully Responsive:** Built with CSS Grid and Flexbox to ensure the game looks great on smartphones, tablets, and desktops.

---

## 🛠️ Technologies Used

This project was built to practice interactive front-end development and state management:

* **HTML5:** Semantic markup structure for accessibility and clear DOM targeting.
* **CSS3:** * **Variables (Root):** Centralized color management for a consistent theme.
    * **Flexbox & Grid:** Used for precise alignment of game controls and scoreboards.
    * **Advanced Styling:** Implementation of `@keyframes` for the title glow and `box-shadow` for the soft UI depth.
* **JavaScript (ES6+):** * **Game Logic:** Randomization algorithms to simulate the machine's choice.
    * **DOM Manipulation:** Real-time updates of text, images, and score counters.
    * **Conditional Logic:** Comparison logic to determine winners, losers, or draws for every round.
